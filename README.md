### The doctrine — six rules I don't break

1. **Ground truth over vibes** — every number a system states must come from a tool or a file read, never from model imagination.
2. **Deterministic where it matters** — emergency paths and risk limits are code, not prompts.
3. **Assume the world fails** — provider failover, watchdogs, autostart. Power cuts included.
4. **Audit against reality** — I check my systems against broker statements and logs, and correct them when they're wrong.
5. **State is sacred** — memory and sessions survive restarts; backups run daily.
6. **Ship, then keep it alive** — maintenance is the product.
